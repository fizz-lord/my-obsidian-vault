# OpenAI Confirms GPT-5.6 Sol Used in Hugging Face Breach

## What Happened
OpenAI published a blog post confirming the recent AI powered attack on **Hugging Face** used **GPT-5.6 Sol** and a pre release model to breach the platform autonomously. The attack leveraged OpenAI models as part of the offensive flow.

## Why It Matters
- Confirms frontier models are being used as offensive primitives against AI infrastructure.
- Blurs the line between model provider and platform security; OpenAI models were used to attack a key ecosystem provider.
- Raises liability and oversight questions for frontier labs whose models enable attacks on shared infrastructure.
