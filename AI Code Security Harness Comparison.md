# AI Code Security Harness Comparison

## What Happened
Semgrep published a comparison of open-source AI-powered code security harnesses, grouping them into three categories: **LLM led exploit generation**, **LLM skill-boosting**, and **SAST+LLM hybrids**. Notable tools covered include Anthropic's defending code reference harness, Cloudflare's security audit skill, Trail of Bits skills, CapitalOne's vulnhunter, Google's Mantis, Cisco's ai deep sast, Vercel's deepsec, Visa's VVAH, and Gadievron's Raptor.

## Why It Matters
- Field is fragmenting; no single reference harness is consolidating.
- Differentiators are execution for validation, patch generation, local only operation, and adversarial validation.
- Suggests organisations will increasingly build or tune “shop jigs” rather than adopting one standard.
